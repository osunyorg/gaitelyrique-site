Un grand tournoi s’organise autour de cette œuvre, à la frontière entre le jeu vidéo, le sport et l’art. Venez dès 14h pour vous entrainer à ce jeu virtuel... A 15h30, les meilleurs coureurs sont sélectionnés pour participer au grand tournoi : quart de finale, demi-finale, finale, que le meilleur gagne ! Le grand tournoi est animé par un DJ set + VJ set. C'est un peu notre boum de l'après-midi, et tout le monde peut venir danser !

14h > 16h
Open qualification (Hyper Olympic Game Contest)
Guillaume Evrard, Dj set
16h
Final phase (Hyper Olympic Game Contest)
SpinTronic, Live 8bit micromusic Gameboy
KiK brigade neurale, VJing

Programmation : Loïc Horellou
Dalles : Charlotte Charbonnel
