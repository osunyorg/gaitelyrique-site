Le GML (Graffiti Markup Language) est un format de fichier qui permet de d'enregistrer un tracé et les données liées à sa durée d'exécution, permettant ainsi sa reproduction numérique, sur écran ou bien en impression 3D.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Le GML fut inventé par le GRL notamment dans une optique de documentation de ces différents et éphémères calligraphies en permettant d'enregistrer un tracé au format numérique.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Pour le mois de juillet et dans le cadre de sa résidence "laboratoire ouvert", le GRL France propose d'approfondir ce type de language et ses moyens de captation/enregistrement. La conférence de juillet sera donc axée sur la questions du style calligraphique dans le tag (époque, origines...).

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Pour ce faire de nombreux acteurs seront invités pour débattre, expliquer et performer autour de cette question.
