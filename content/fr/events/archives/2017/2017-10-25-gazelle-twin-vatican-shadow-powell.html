De l'électro-industrielle de Gazelle Twin, à la techno agressive de Vatican Shadow en passant par l'electro-punk de Powell, plusieurs nuances de noir se relaient ce soir dans la grande salle pour le retour de la biennale Némo à la Gaîté.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Gazelle Twin

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-


Accompagnée de Jez Bernholz, la productrice de Brighton Elisabeth Walling aka Gazelle Twin nous présente ici son nouveau spectacle : Kingdom Come. Spectacle et non simplement live, car ici la musique se double de performances scéniques, et de projections qui immergeront la grande salle dans l'univers des deux artistes.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Vatican Shadow

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-


Egalement reconnu sous le nom de Prurient, le New Yorkais Dominick Fernow se présente ce soir sous son nouvel alias : Vatican Shadow. Un alias derrière lequel se cache un artiste aux lives énervés.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Powell

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-


Effusion sonore électro-punk, doublée d'une performance vidéo signée Weirdcore, New Beta AV Show, le nouveau live de Powell saura sustenter les plus exigeants amateurs du genre.
